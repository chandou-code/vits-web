VITS接口文档

www.纯度.site:5000/run?text=你要合成的文字&id_speaker=119&length=2.25&noise=0.32&noisew=0.232&lang=zh

【说明】length是长度 越大越慢，noise是情感起伏大小，noisew是音素发音长度，lang可以是auto(混合)或者ja(日语)，GET，单次最多600字。

【说明】为了节约内存，初次合成的时间包含开启vits的耗时，有点久，往后的三分钟内合成语音都是正常GPU速度，三分钟内没有新的请求就会重启服务释放内存

www.纯度.site:5000/models

【说明】返回模型列表

www.纯度.site:5000/if

【说明】接口是否可用 如果可用返回True 不可用则404无法响应
